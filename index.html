ttyplayer.js

ttyplayer 是一个浏览器端的 tty 播放器，使用它录制和分享你的命令行。

如何使用

安装 ttyrec

在使用 ttyplayer 之前，你需要使用 ttyrec 录制记录文件。如何安装 ttyrec 我就不详细说了，在 Mac 下可以使用 homebrew 安装。

使用 ttyrec

使用如下命令开始进行录制：

ttyrec your.rec

录制完成后，使用 exit 退出。

使用 ttyplayer

把 dist 目录下的 ttyplayer.min.css 和 ttyplayer.min.js 两个文件引入你的项目，然后初始化播放器：

new TTYPlayer({
  parent: document.getElement('player-parent')
}).load('/your.rec')

注意

由于网页宽度限制，ttyplayer 默认宽高为 80 * 20，所以在录制前需要把终端宽高调整到和播放器一致。 你也可以传入 cols 和 rows 控制播放器宽高。 

new TTYPlayer({
  parent: document.getElement('player-parent')，
  cols: 90,
  rows: 30
}).load('/your.rec')
